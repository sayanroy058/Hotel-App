Telegram Bot Setup Guide

Follow these steps to set up Telegram notifications for your hotel

Setup Steps
1 Start the Bot

Open Telegram and search for the bot using the bot token provided by your administrator, or click this link:

Open Bot in Telegram
2 Get Your Chat ID

Send the command /start to the bot. The bot will reply with your Chat ID.

Example response:
"Welcome to the Hotel Management Bot! Your chat ID is: 123456789
Please provide this ID to your hotel administrator to receive notifications."
3 Provide Chat ID to Administrator

Copy the Chat ID number and provide it to your hotel administrator so they can configure notifications for your hotel.

4 Test the Connection

Once configured, your administrator can send a test message to verify the connection is working.

What Notifications Will You Receive?
  • New booking confirmations
  • Check-in/check-out alerts
  • Booking cancellations
  • Payment status updates
  • System alerts and reminders
Troubleshooting
Bot not responding?
  • Make sure you've clicked "Start" in the bot chat
  • Check if the bot token is correct
  • Try sending /help command
Not receiving notifications?
  • Verify your Chat ID is correctly configured
  • Ask admin to send a test message
  • Check if notifications are enabled in Telegram
Need help?

Contact your system administrator or hotel management for assistance.

Privacy & Security

Your Chat ID is only used to send hotel-related notifications. We don't store or access any other Telegram data. You can stop receiving notifications at any time by blocking the bot.
